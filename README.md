# Archives Text Kit
Notre Dame's standard archival processing text expansions for use with ArchivesSpace 

## Notes:
- Expansions are organized into 3 groups: Expansions by EAD tag (includes one file of expansions per tag); dates and markup (includes expansions for today's date in DACS and EDTF, expansions for foramtting wrapper tags for use inside and outside of ASpace, universal ASpace platform codes); Expansions for local names and codes (record numbers for form genre subject terms, local names)
- Notre Dame's custom values are present within files in the local_names_codes folder, and in variables.yml
- Variables are configured to encode institutional metadata for several repositories located within an academic library system, as well as a standard institutional records retention length in years
- Pressing alt + space will retrieve a list of all expansions installed on the local machine
