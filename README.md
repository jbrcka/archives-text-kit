# Archives Text Kit
A set of standard archival processing text expansions for use with ArchivesSpace

## Notes:
- Expansions are organized into 3 groups: Expansions by EAD tag (one file of expansions per tag); dates and markup (expansions for today's date in DACS and EDTF, expansions for foramtting wrapper tags for use inside and outside of ASpace, universal ASpace platform codes)
- Notre Dame's custom values are present within files in the local_names_codes folder, and in variables.yml
- Variables are configured to encode institutional metadata for several repositories housed within a library system in an academic library
- Pressing alt + space will retrieve a list of all expansions installed on the local machine
